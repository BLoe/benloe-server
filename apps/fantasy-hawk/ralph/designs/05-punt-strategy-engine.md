# Punt Strategy Engine - Design Specification

## Overview

**Primary Use Case**: "What build am I currently running? Should I commit to punting assists?"

## Component Specifications

### 1. Current Build Detection

**Component**: `CurrentBuildCard`
**Test ID**: `data-testid="punt-current-build"`

Shows detected punt strategy with confidence percentage and explanation.

### 2. Category Ranks Display

**Component**: `CategoryRanksChart`
**Test ID**: `data-testid="punt-category-ranks"`

Horizontal bar chart showing your rank (1-12) in each category. Flag weak categories.

### 3. Build Archetypes

**Component**: `BuildArchetypeCards`
**Test ID**: `data-testid="punt-archetypes"`

Cards for common punt builds:
- Punt AST
- Punt FT%
- Punt 3PM
- Punt 3PM+PTS
- Punt FG%+TO (rare)

Each shows match percentage and brief description.

**Data TestIDs**:
- `punt-archetype-{name}`
- `punt-archetype-match-{name}`

### 4. What-If Simulator

**Component**: `PuntSimulator`
**Test ID**: `data-testid="punt-simulator"`

Dropdowns to select categories to punt. Shows projected impact.

**Data TestIDs**:
- `punt-simulator`
- `punt-simulator-select-{index}`
- `punt-simulator-results`

### 5. Correlation Matrix

**Component**: `CategoryCorrelationMatrix`
**Test ID**: `data-testid="punt-correlation-matrix"`

Heatmap showing which categories tend to come together.

## Data TestIDs Summary

- `punt-current-build`
- `punt-category-ranks`, `punt-category-rank-{category}`
- `punt-archetypes`, `punt-archetype-{name}`
- `punt-simulator`, `punt-simulator-select`
- `punt-correlation-matrix`
